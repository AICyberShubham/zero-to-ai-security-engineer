# ğŸ”µ 2. Building LLM Applications

## ğŸ“˜ A) Retrieval-Augmented Generation (RAG)
RAG = **LLM + your own documents**  
Used for:
- chat with PDFs  
- log investigation  
- knowledge bases  
- resumes  
- documentation bots  

### RAG Pipeline
1. Convert documents â†’ text  
2. Make embeddings  
3. Store vectors in a DB  
4. Retrieve relevant chunks  
5. Feed into model  

### Example (Simple RAG Retrieval)
```python
from sentence_transformers import SentenceTransformer
import chromadb

client = chromadb.Client()
model = SentenceTransformer('all-MiniLM-L6-v2')

text = "Network logs show suspicious activity"
embedding = model.encode(text).tolist()

collection = client.create_collection("security_logs")
collection.add(documents=[text], embeddings=[embedding], ids=["1"])
```
